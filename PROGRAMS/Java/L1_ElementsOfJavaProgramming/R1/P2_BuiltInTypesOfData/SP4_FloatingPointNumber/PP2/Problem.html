Practice Program

Write a program Quadratic.java to find the roots of the equation axx + bx + c. Since the equation is xx, hence there are 2 roots. The 2 roots of the equation can be found using a formula
delta = bb - 4ac
Root 1 of x = (-b + sqrt(delta))/(2a)
Root 2 of x = (-b - sqrt(delta))/(2*a)
Take a, b and c as input values to find the roots of x.
